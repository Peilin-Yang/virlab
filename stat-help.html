score: the variable store the grade of evaluated document. The whole function will return it eventually.
tf[i]: term frequency for each query terms in this document.
DF[i]: Document frequency which contain each query terms.
qf[i]: query term frequency;
termPro[i]: query term probability within the collection
docN: total number of documents in the collection
docLength: document length of the evaluated document
docLengthAvg: average document length of the whole collection
termN: total unique terms in the collection
collectionN: total terms in the collection (collection size)
MaxDF: The highest document frequency for a term in the collection
TFC[i]: The total number of occurrence of a query term in the collection.
